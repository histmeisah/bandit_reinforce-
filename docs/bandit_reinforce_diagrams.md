## 5. NeuralUCB Algorithm Flow (NeuralUCB算法流程)

```
┌──────────────────────────────────────────────────────────────────────────────┐
│                         NeuralUCB Algorithm                                  │
└──────────────────────────────────────────────────────────────────────────────┘

                    INITIALIZATION
                          │
                          ▼
    ┌────────────────────────────────────────────────────────┐
    │  For each arm i = 1, ..., K:                           │
    │    • Neural Network: f_i(x; θ_i) → μ_i                 │
    │    • Inverse Covariance: A_i⁻¹ = (1/λ) · I             │
    │    • Replay Buffer: D_i = {}                           │
    └────────────────────────────────────────────────────────┘
                          │
                          ▼
┌──────────────────────────────────────────────────────────────────────────────┐
│                         ARM SELECTION                                        │
├──────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│     Input: context x (problem embedding)                                     │
│                          │                                                   │
│                          ▼                                                   │
│     ┌────────────────────────────────────────────────────────────┐          │
│     │  For each arm i:                                            │          │
│     │                                                             │          │
│     │    ┌──────────────┐                                         │          │
│     │    │   x ────────►│ Neural Net f_i                          │          │
│     │    │              │     │                                   │          │
│     │    │              │     ├──► μ_i = f_i(x)  (predicted reward)│         │
│     │    │              │     │                                   │          │
│     │    │              │     └──► φ_i = features (128-dim)       │          │
│     │    └──────────────┘                                         │          │
│     │                                                             │          │
│     │    ┌──────────────────────────────────────────────────┐    │          │
│     │    │  Confidence Bound:                                │    │          │
│     │    │                                                   │    │          │
│     │    │   σ_i = α · √(φ_iᵀ · A_i⁻¹ · φ_i)                │    │          │
│     │    │                                                   │    │          │
│     │    │  UCB Score:                                       │    │          │
│     │    │                                                   │    │          │
│     │    │   UCB_i = μ_i + σ_i                               │    │          │
│     │    │         ─────   ────                              │    │          │
│     │    │       exploitation  exploration                   │    │          │
│     │    └──────────────────────────────────────────────────┘    │          │
│     └────────────────────────────────────────────────────────────┘          │
│                          │                                                   │
│                          ▼                                                   │
│     ┌────────────────────────────────────────────────────────────┐          │
│     │  Select: i* = argmax_i (UCB_i)                             │          │
│     └────────────────────────────────────────────────────────────┘          │
│                          │                                                   │
│                          ▼                                                   │
│                    Return i*                                                 │
└──────────────────────────────────────────────────────────────────────────────┘
                          │
                          ▼
┌──────────────────────────────────────────────────────────────────────────────┐
│                           UPDATE                                             │
├──────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│     Input: arm i, context x, reward r                                        │
│                          │                                                   │
│                          ▼                                                   │
│     ┌────────────────────────────────────────────────────────────┐          │
│     │  1. Store experience:                                       │          │
│     │     D_i ← D_i ∪ {(x, r)}                                    │          │
│     └────────────────────────────────────────────────────────────┘          │
│                          │                                                   │
│                          ▼                                                   │
│     ┌────────────────────────────────────────────────────────────┐          │
│     │  2. Update inverse covariance (Sherman-Morrison):           │          │
│     │                                                             │          │
│     │     φ = f_i.get_features(x)                                 │          │
│     │                                                             │          │
│     │                    A_i⁻¹ · φ · φᵀ · A_i⁻¹                   │          │
│     │     A_i⁻¹ ← A_i⁻¹ - ─────────────────────                   │          │
│     │                      1 + φᵀ · A_i⁻¹ · φ                     │          │
│     │                                                             │          │
│     │     (O(d²) instead of O(d³) matrix inversion)               │          │
│     └────────────────────────────────────────────────────────────┘          │
│                          │                                                   │
│                          ▼                                                   │
│     ┌────────────────────────────────────────────────────────────┐          │
│     │  3. Periodic network training (every update_freq):          │          │
│     │                                                             │          │
│     │     if t % update_freq == 0:                                │          │
│     │         Sample batch B from D_i                             │          │
│     │         L = MSE(f_i(x_B), r_B) + λ·‖θ_i‖²                   │          │
│     │         θ_i ← θ_i - η·∇L                                    │          │
│     └────────────────────────────────────────────────────────────┘          │
│                                                                              │
└──────────────────────────────────────────────────────────────────────────────┘
```
